Zomato Geospatial Data Analysis

# Project Overview
This project analyzes Zomato restaurant data to uncover location-based trends in ratings, pricing, and restaurant distribution. The analysis focuses on understanding how geography influences customer preferences and business performance.

# Objective
- Analyze restaurant distribution across different locations  
- Identify trends in ratings and pricing  
- Understand relationship between cost, location, and customer ratings  

# Tools & Technologies
- Python  
- Pandas  
- Matplotlib / Seaborn  
- Jupyter Notebook  

# Dataset
Zomato restaurant dataset containing information on location, ratings, cost, and cuisines.

# Steps Performed
1. Data Cleaning  
   - Handled missing values  
   - Cleaned and formatted cost and location columns  
   - Removed duplicates  

2. Exploratory Data Analysis (EDA)  
   - Analyzed restaurant distribution by area  
   - Studied rating patterns across locations  
   - Explored cost vs rating relationships  

3. Data Visualization  
   - Created location-based visualizations  
   - Compared ratings and pricing across areas  
   - Identified high-performing regions  

# Key Insights
- Certain locations have a higher concentration of highly-rated restaurants  
- Higher cost does not always guarantee better ratings  
- Popular areas show increased competition among restaurants  

# Conclusion
The analysis highlights the impact of location on restaurant success and customer behavior, providing useful insights for business expansion and pricing strategies.
